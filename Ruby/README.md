# Gilded Rose Ruby

## Getting Started

* Install Ruby 3.1.2
  * For rbenv: `rbenv install 3.1.2`
* Install `bundler`
  * `gem install bundler`
* Install the dependencies
  * `bundle`

To run the tests: `bundle exec rspec spec`

## Where to Start

Begin refactoring the existing `lib/gilded_rose.rb` class.
